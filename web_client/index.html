    Space Station Alpha - Command Interface

    Available commands for station operations:

    • help - Display available commands
    • look - Scan current location
    • go [direction] - Move to another section (north, south, east, west)
    • inventory or i - View personal inventory
    • examine [item] - Inspect an item more closely
    • take [item] - Collect an item into inventory
    • drop [item] - Release an item from inventory
    • use [item] - Utilize an item's function
    • scan - Perform biometric and environmental scan
    • stats - View personal biometric statistics
    • say [message] - Communicate verbally
    • quit - Terminate station interface connection

    For detailed command specifications, enter 'help' in the command input.
